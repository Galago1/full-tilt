## Accessibility
To ensure assistive technologies like screen readers can identify and announce button groups correctly, assign the appropriate role attribute and an explicit label. Use role="group" for button groups or role="toolbar" for button toolbars. Then, use aria-label or aria-labelledby to provide a label.
